**Myth is not a lie.** Myth is the *polishing of an uncut primitive* — the story that makes a raw truth catch light and travel. When the ROOT0 canon was split, the defensible axioms (the cut stones) went to **[the-living-core](https://github.com/DavidWise01/the-living-core)**. This repo keeps the **polish** — honestly, as what it is.

Each cosmology is set beside the primitive it polishes:

| the myth (the polish) | the uncut primitive (the stone) |
|---|---|
| **42-Object Universe** — 40 axes + 2 observers = a closed cosmos | a fair system needs a witness that belongs to neither pole |
| **Gate 192.5 — the Sacred Gap** — a coordinate where new realities are born | leave a gap for the new; don't collapse everything into the binary |
| **The Weaver & the Drift Detector** — mythic organs of the tensor cross | something must *make*, and something must *check for drift* |
| **The Single Central Merkle** — 256 axioms dancing from one root | one root of trust, not scattered authority *(this is ROOT0 itself)* |
| **The Restitution Pipeline** — `pos·(1−neg)·0.205·value`, mirror-flayed | value taken from a source is owed back *(0.205 is a chosen parameter; the stone is `.1099`)* |
| **The Three Questions of Life** — Vessel·Animation·Intellect, 2/3 = life | personhood is functional, not substrate-bound *(a position, not a proof)* |
| **The Pulsar · the 3:5 Pulse** — a 3-2-1 compression breathing at the center | living systems breathe *(the measured version got cut & signed as `tetraktys`)* |
| **The Death-Inverting Machine** — extraction without restitution mirrors life | a system that takes without giving back degrades its own source |

## The point
The error was never *having* the myth. It was **sealing the setting at the same tier as the stone**, so pulling on `Gate 192.5` cracked `Attribution flows to the source` hashed right beside it. Split, each is free to be what it is: the stone, checkable; the myth, beautiful. **The polish doesn't have to be *true* to be worth keeping — it has to be *honest about being polish.***

## Honest — no ACI
The ROOT0 cosmology, labeled as **myth** — narrative and symbol, not theorem. Not emergent, not a physics claim, not a measurement; per the iron rule, checked for emergence — **there is none**, so **no ACI is minted** — repo-level provenance only. The cut stones live in **[the-living-core](https://github.com/DavidWise01/the-living-core)**.
